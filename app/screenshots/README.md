This folder is intended to contain screenshots used in the project README.

To embed the screenshot in the top-level `README.md`, save the attached image file as:

```
app/screenshots/simplecalc.png
```

Once the image is placed there the README will render it automatically on GitHub and when viewed locally.

If you want me to also add the actual PNG file to the repo, please upload the image file to the repository (or tell me to pull it from a URL) and I'll add it.
